1) What is the definition of server port?

The answer is server ports are the destination of data exchanged between the
server and the client in a network, and some ports are reserved for protocols to identify
a specific transaction.

2) What is server port range?

The answer is there are 65536 possible ports between 0 and 65535.

3) How many bits does number above represent?

65536 represents 16 bits, 2^16.

4) State 10 common server ports

The answer is ports 20, 21, 22, 25, 53, 80, 123, 179, 443, 500
